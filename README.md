##  📍 Pré-requisitos
 
```sh
- ruby > 5.5.0
- rspec > 3.4.0
- git > 2.36.1
- cucumber > 8.0.0
- capybara > 3.37.1
- site_prism > 3.7.3
- selenium-webdriver > 4.3.0
```

# ⏳ Instalação:
## ⚠️ Importante:
  chromeDriver
```sh
- Baixar a ultima versão do Google Chrome
- Extrair o zip no diretório: C:\ruby31-x64\bin
```

intalação necessárias (gems)

```sh
- cd /RealityStone_Everson_Tibolla_Compass_Sprint6
```
